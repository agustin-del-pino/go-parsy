# Examples

The example in `main.go` demonstrates how to print on the terminal the **node type** of a source code from a string. 

Run this command
```shell
go run main.go
```

The following output has to appear as result

```shell
*ast.File
*ast.Ident
*ast.GenDecl
*ast.ValueSpec
*ast.Ident
*ast.BasicLit
*ast.ValueSpec
*ast.Ident
*ast.BasicLit
```
